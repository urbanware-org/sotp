    Project

        SOTP
        Version 1.0.3 (based on Python 2.x framework)
        Copyright (C) 2017 by Ralf Kilian

        GitHub: https://github.com/urbanware-org/sotp
        GitLab: https://gitlab.com/urbanware-org/sotp

    Definition

        The SOTP project is a simple tool for reading and writing config
        files. It allows to read as well as modify existing config files and
        also to create new ones.
